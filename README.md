# Dachuang-MS（大创项目管理平台）

本仓库包含：

- `backend/`：Django + DRF 后端（默认 `http://localhost:8000`）
- `frontend/`：Vue 3 + Vite 前端（默认 `http://localhost:3000`）
- `dachuang_db_full.sql`：PostgreSQL 数据库结构与测试数据（用于快速初始化）
- `dachuang_db_schema.sql`：PostgreSQL 数据库结构（无测试数据）

解压项目压缩包后，按文档完成数据库配置与本地运行：

- 项目运行说明：`docs/SETUP.md`
- 数据库文件与配置说明：`docs/DB.md`
